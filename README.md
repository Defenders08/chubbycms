## Хранение данных
**Статьи**
Все статьи хранятся в txt файлах, тобишь flat‑file система.
 
**Медиа**
Всё медиа хранится в /files. Есть три категории:
 - **/files/img** - Медиа
 - **/files/music** - Музыка
 - **/files/files** - Остальное

**Парсер**
Файл `parser.php` содержит небольшой **кривой** парсер.
 - [tab][/tab] - Блок cite
 - [h3][/h3] - H3 заголовок
 - [text][/text] - Костыль для [img_little]
 Списки реализованы так:

>   [ul]
> 	   [li]пункт[/li] 
> 	   [li]пункт2[/li] 
> [/ul]

Картинки:

> [tab] 
	> [img_little](files/img/2025-11-12 15.40.46.jpg )Описание для
	> картинки[/img_little] 
	> [text] 	
	> Тот самый костыль) 
	> [/text]
> [/tab]
